Entering Primary and Secondary Providers

You can enter a primary provider and secondary providers in the Encounter Frame. The primary provider is designated by the word "primary" in parenthesis.

Also, you cannot remove a primary provider until you have designated a new primary provider.

1.     In the Encounter Frame, select the Visit Type tab.

2.     To assign a new provider, select the provider's name under Available Providers and then select Add. You may do this several times to move more than one provider onto the Current Providers list.

3.     To designate the primary provider, select the provider under Current Providers and select Primary.

4.     To remove providers from the Current list, select the provider name and click Remove.

5.     When finished, select OK.
