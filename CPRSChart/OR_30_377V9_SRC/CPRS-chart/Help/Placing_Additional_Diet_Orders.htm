Placing Additional Diet Orders

To place an additional diet order, follow these steps:

1.     Select the Orders tab.

2.     Select Diet in the Write Orders list box.

Note:      The Encounter Information dialog appears if no encounter information has been entered.

 

3.     Select the Additional Order tab.

4.     Type in (free text) the order that you want to place.

5.     Select Accept Order.

6.     When finished, click Quit.

Note:      The order must be signed before it is sent. You can either sign the order now or wait until later.
