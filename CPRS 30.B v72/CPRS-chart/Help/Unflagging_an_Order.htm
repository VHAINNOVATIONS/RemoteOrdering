Unflagging an Order

You can remove the flag from an order very simply. You can enter a reason for removing the flag, but CPRS does not require you to enter a reason.

In the order's detailed display, CPRS will record the name of the person who removed the flag and the date and time that person removed it.

To unflag an order, use these steps:

1.     Select the Orders tab.

2.     Select the order you want to unflag.

3.     Select Action | Unflag.

4.     (Optional) Enter a reason, if desired.

5.     Select OK.
